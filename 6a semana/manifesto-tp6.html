Abstract

Foi desenvolvida uma aplicação de gestão de tarefas com recurso ao NodeJS e à framework Pug.

A presente tarefa serviu também para solidificar os conhecimentos relativos ao elemento HTML, form, e às suas diferentes formas de utilização.

Para assegurar a persistencia dos dados, é utilizada uma API de leitura e escrita de ficheiros no formato JSON.

A aplicação final apresenta todas as quatro funções básicas, CRUD.
